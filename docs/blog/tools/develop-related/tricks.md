# 小技巧汇总

## Mac 终端命令行用 Vscode 打开文件夹。

```bash
vim ~/.bash_profile
export PATH=/Applications/Visual\ Studio\ Code.app/Contents/Resources/app/bin:$PATH
#保存退出
source ~/.bash_profile
code . #当前目录就被打开了。
```
